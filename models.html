class networks.models.ProjectionMapping(*args, **kwargs)[source]¶

A rule about how to collapse Node instances in a Network.

For example, we may wish to collapse all Node representing people into a smaller set of Node representing the institutions with which they are affiliated. In that case, primaryNode would refer to a NodeType representing institutions, and secondaryNode would refer to a NodeType representing persons.

Attributes

primaryNode : ForeignKey relation to a NodeType. The Node to subsume secondaryNodes.
secondaryNodes : ManyToMany relation to NodeType. These Node are subsumed by primaryNode.
